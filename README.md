# Know Your Schedule – React App

A simple appointment management web application built using React.
Users can add appointments, mark important ones with a star, and filter starred appointments.

## Live Demo

https://knowyoursched.ccbp.tech/

## Features

* Add new appointments with title and date
* Star important appointments
* Filter to view only starred appointments
* Date formatting using `date-fns`
* Clean and responsive UI

## Tech Stack

* React.js
* JavaScript (ES6)
* CSS
* date-fns

## Project Structure

src
components
Appointments
AppointmentItem

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run the Application

```bash
npm start
```

If you are using **Node.js 17+**, run:

```powershell
$env:NODE_OPTIONS="--openssl-legacy-provider"
npm start
```

Then open:

```
http://localhost:3000
```
